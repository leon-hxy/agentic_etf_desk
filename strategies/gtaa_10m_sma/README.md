# GTAA 10M SMA

Repo-only ETF research template for a global tactical ETF allocation using a monthly 10-month simple moving average filter. Risk assets are held only when price is above the 10-month average; otherwise the model moves to a defensive allowed ETF.

This template produces research and backtest outputs only. This is research advice, not automatic order placement. 最终交易由用户手动决定。
