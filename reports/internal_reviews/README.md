# Internal Reviews

This directory stores Codex self-review notes for small stages.

Codex self-review is required for small stages before handoff is finalized.
Small stages should record scope, changed files, tests, risk notes, and whether
the stage stayed repo-only.

Do not request ChatGPT review for small stages.

Do not run Computer Use for internal review delivery.

Every internal review must preserve ETF-only, manual final trading, no broker
write interface, no automatic trading behavior, and no secret publication.
