# Major Reviews

This directory stores major-stage review packages and user-facing review request
materials.

Major stages use manual ChatGPT review. Codex prepares public repo materials,
including the public GitHub URL, review target commit, repo-relative handoff
paths, repo-relative review request paths, test commands, and test results.

manual ChatGPT review is reserved for major stages.

Do not use Computer Use to send review requests to ChatGPT.

Do not open ChatGPT automatically.

Do not send ChatGPT prompts automatically.

The user decides whether and when to request manual ChatGPT review for a major
stage.

Every major review must preserve ETF-only, manual final trading, no broker
write interface, no automatic trading behavior, and no secret publication.
