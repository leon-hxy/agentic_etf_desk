# OpenClaw Draft Configs

These files are repo-only drafts for a future approved OpenClaw setup. They are not applied to real runtime config and 不得应用到真实 ~/.openclaw.

The draft keeps workspace isolation and minimum permissions. Agents may only produce research, backtests, risk review, reports, manual recommendation tickets, and portfolio journals.

No agent receives broker write permissions. No agent may place trades. 最终交易由用户手动决定。
